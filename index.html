Welcome

The aim of this part of the workshop is to introduce phylogenies as potential tools for conservation. We’ll first examine what phylogenies are (what do they describe? how do we “read” them?), then we’ll check out how to read them into R, and manipulate them as data objects (e.g. how to subset them, plot different parts, join them to useful data etc.)

Installing relevant packages

There are lots of hundreds of packages avaliable in R, and dozens focussing on phylogenetic analysis. We’ll only be using a few of them, but you can install a large number in one go by using R’s “task views”. First you need to install a package called “ctv”, then you can install all the packages that are focussing on a partiuclar area with a single command. You can find task views at: http://cran.r-project.org/web/views/. The task views for “Graphics”, “Reproducible Research” might be useful.

install.packages("ctv")
library("ctv")
install.views("Phylogenetics")

If you’d rather just install a select few packages, the ones we’ll be using today are ape, caper, geiger and picante. Install them in the usual way e.g. install.packages("ape") and load them with library("ape").
